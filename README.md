# ❌⭕ Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🔹 Live Demo

👉 https://asishkumar91.github.io/tic-tac-toe/

## 🔹 Features

* 👥 Multiplayer mode (Player vs Player)
* 🤖 Play against computer (Bot)
* 🧠 Smart bot using recursive logic
* 🎯 Real-time win/draw detection
* 🎨 Clean and user-friendly interface

## 🔹 About the Bot

The computer player is designed using a recursive approach to decide its moves.
It evaluates possible outcomes and tries to make optimal decisions during gameplay.

## 🔹 Technologies Used

* HTML
* CSS
* JavaScript

## 🔹 Project Structure

* `index.html` → Game layout
* `TOE.css` → Styling and UI
* `TAC.js` → Game logic and bot behavior
